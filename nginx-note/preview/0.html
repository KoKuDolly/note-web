安装

mac

安装命令

brew install nginx

更新nginx命令

brew upgrade nginx

安装目录查找命令

which nginx

# 位置
# /usr/local/bin/nginx

mac 配置文件目录

/usr/local/etc/nginx/nginx.conf

mac启动nginx命令

sudo nginx

重新启动nginx命令

sudo nginx -s reload

关闭nginx

nginx -s stop

nginx -s quit
